# aws-durable-functions-research

Research repository for evaluating AWS Lambda Durable Execution in the context of resilient serverless workflows.

## Project Scope

This repository contains the implementation, experiments, and documentation for a multi-phase doctoral research project focused on durable execution, workflow reliability, retry semantics, concurrency, and state management.

## Repository Structure

- `phase1/`: Durable counter implementation and experimental evaluation
- `phases/phase2/`: Shared-state and extended experiments
- `phases/phase3/`: Advanced workflow scheduling and orchestration analysis

## Phase 1

Phase 1 includes:
- Durable counter implementation
- Normal execution experiments
- Controlled failure and retry analysis
- Concurrency experiments
- LaTeX report and screenshots

## Technology Stack

- AWS Lambda Durable Execution
- Python
- AWS Console experiments
- LaTeX documentation
